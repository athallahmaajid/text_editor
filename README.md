# text_editor
text editor with tkinter

# Requirements
 1. Tkinter

## How to install
  Windows:
  ```
    http://www.tkdocs.com/tutorial/install.html
  ```
  Debian/Ubuntu:
  ```
  sudo apt-get install python3-tk
  ```
  Mac:
  ```
  brew install tcl-tk

  brew link tcl-tk --force
  ```
# How to run

```
python text_editor.py
```
